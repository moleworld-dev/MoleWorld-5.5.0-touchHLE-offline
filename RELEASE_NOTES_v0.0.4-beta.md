# 🐭 摩尔庄园HD 移动版 v0.0.4 beta · VIP解锁 + 窗口自由缩放 + 工程大一统

> 这一版功能给得很足:**VIP 功能解锁(修好了闪退)**、**窗口能自由改大小 / 锁定比例**、**iPhone 版切换**;同时把两个工程隐患一起解决了 —— **安卓以后更新不再签名冲突**、**全平台版本号大一统**。

继 v0.0.3 beta(离线贝壳逻辑修正)之后,本次 **v0.0.4 beta** 覆盖 **macOS / Windows / Linux / Android 四大平台**,**无需服务器、无需登录、解压(或安装)即玩**。

---

## 🚨 安卓用户必读(这一版要卸载重装一次)

本版起,安卓改用了**固定签名**(以前每次构建都是随机签名,所以你**装新版总报「签名冲突」、装不上**)。代价是:

- **装过旧版(v0.0.1/2/3)的安卓用户,装这个 v0.0.4 仍会签名冲突 —— 请先把旧版卸载,再装新版**(卸载会清掉游戏在手机里的存档)。
- **但从这一版往后,所有更新都能直接覆盖安装、永不再冲突。** 这是一次性的阵痛,换来以后顺滑更新。

桌面端(mac/Win/Linux)不受影响,照常解压即玩。

---

## ✨ 本版更新(v0.0.4 beta)

- 👑 **VIP 功能解锁(修复闪退)**:开「强制VIP」一进 VIP 界面就闪退的 bug 修好了(根因:一个 VIP 等级方法的返回类型被当成了整数指针)。现在按 **T** → 「开关」页 →「强制VIP」开,VIP 专属商店/装扮/功能离线即解锁、**不再闪退**;「VIP等级」封顶 **VIP4**。默认关闭,需手动开启。
- 🖥️ **窗口可自由改变大小 + 锁定比例**:游戏窗口能拖动边角自由缩放、画面自由拉伸铺满;也可选**锁定比例**模式 —— 窗口始终保持游戏原始比例(4:3 / 3:2),画面填满不变形、不留黑边。
- 📱 **iPhone 版切换(mac)**:新增 iPhone 版启动脚本,游戏加载整套 iPhone 横屏美术(480×320)。
- 🏷️ **窗口标题显示版本号**:标题栏显示 `摩尔庄园HD · v0.0.4 beta (构建hash)`,一眼看清版本、便于反馈定位。
- 🔏 **安卓固定签名**:见上方「安卓用户必读」—— 以后更新不再签名冲突。
- 🔢 **版本号大一统**:全平台统一显示 `v0.0.4 beta (短hash)`,去掉了之前安卓上那个迷惑的 `V0.0.1-beta-8-g… UNOFFICIAL` 串;`(短hash)` 是构建追溯号,方便回溯具体是哪次构建。

> 🍎 **macOS 启动脚本**(解压后双击):`启动摩尔庄园.command`(自由拉伸)/ `启动摩尔庄园-锁定比例.command`(锁定 4:3)/ `启动摩尔庄园-iPhone版.command`(iPhone 480×320)。

## 📦 各平台下载与玩法

<details>
<summary>👉 <b>点击展开</b> —— macOS / Windows / Linux / Android 各平台下载与一键玩法</summary>

> 桌面三端均为**未签名**版本,首次启动时系统可能拦截,按下面步骤选择「信任 / 打开」即可,属正常现象。

### 🍎 macOS(Apple Silicon)

**下载:`MoleWorld_v0.0.4_beta_macOS_AArch64.zip`**

1. 解压 zip。
2. **右键**点击「摩尔庄园.app」→ 选择「**打开**」(请勿直接双击)。
3. 首次会弹出未签名提示,在弹窗里再点一次「**打开**」即可通过 Gatekeeper。
4. 进入游戏 🎮(想锁比例 / iPhone 版,改用解压目录里对应的 `.command` 启动脚本)

> 如果右键打开仍被拦截,可在终端执行:`xattr -dr com.apple.quarantine 摩尔庄园.app`

---

### 🪟 Windows(10 / 11 · 64 位)

**下载:`MoleWorld_v0.0.4_beta_Windows_X86_64.zip`** → 解压 → 双击 **`Run-MoleWorld.bat`** → 进游戏 🎮

---

### 🐧 Linux(64 位桌面)

**下载:`MoleWorld_v0.0.4_beta_Linux_X86_64.tar.gz`** → 解压进文件夹 → 双击 **`启动游戏.sh`**(GNOME 需右键「以程序运行」)→ 进游戏 🎮。需 OpenGL / SDL2 运行库。

---

### 🤖 Android(arm64)

**下载:`MoleWorld_v0.0.4_beta_Android_AArch64.apk`**

1. **装过旧版的请先卸载**(见顶部「安卓用户必读」),再安装本 apk(debug 签名,需允许「未知来源 / 安装未知应用」)。
2. 打开 App,**直接进入游戏**——游戏本体已内置 🎮

## ⚠️ 已知限制

以下内容属于**离线复活的天然限制**(原本依赖已死的服务器),**并非 bug**:

- **17 个季节联网活动**:美术资源从服务器下载、本地不存在且需联网校验,无法进入。
- **黄金岛(加勒比寻宝)**:核心美术 31/32 张是服务器下载的、本地及四个旧版全无,**离线救不回**,点了干净退回村庄。
- **多人 / 好友 / 社交 / 排行榜 / 广告 / VIP 在线奖励 / 充值**:依赖服务器,离线不可用。
- **「等级不涨」**:5.5.0 新增的 `curLevel` XOR 混淆在运行时层处理仍有问题,可用**修改器(按 `T`)直接设定等级**绕过。
- **retina 高清美术**:模拟器 `UIScreen.scale` 暂写死 1.0,游戏只加载非 retina 美术(iPad 1024×768 是当前最清晰路径);retina 支持是后续独立课题。
