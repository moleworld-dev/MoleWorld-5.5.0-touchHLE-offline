# 🛠️ 摩尔庄园HD 移动版 v0.0.2 beta · 离线改名修复版

> 上一版(v0.0.1 beta)让庄园在现代设备上离线复活了;这一版,我们把「修改庄园名字」也修通了 —— **包括中文名**。

继 v0.0.1 beta 首发四平台开箱即玩之后,本次 **v0.0.2 beta** 是一个**修复版**:重点解决离线状态下「修改庄园名字」被网络校验拦住 / 改完闪退的问题,并补齐一批稳定性与体验改进。依旧覆盖 **macOS / Windows / Linux / Android 四大平台**,**无需服务器、无需登录、解压(或安装)即玩**。

---

## ✨ 本版重点:离线改名修通了(含中文)

老游戏的「改名」原本要联服务器校验:离线下要么被「咦,你的设备没有连接网络哦」挡住,要么改完就闪退。本版参考 **2.4.3 旧版「纯本地改名」**的做法,在模拟器层把改名落地到本地存档:

- ✅ **离线就能改庄园名字**,改完**存进存档**,退出重进保持;
- ✅ **中文名也能改**(修复了归档非 ASCII 字符串的崩溃);
- ✅ **不再弹「无网络」、不再闪退**。

## 📦 各平台下载与玩法

<details>
<summary>👉 <b>点击展开</b> —— macOS / Windows / Linux / Android 各平台下载与一键玩法</summary>

> 桌面三端均为**未签名**版本,首次启动时系统可能拦截,按下面步骤选择「信任 / 打开」即可,属正常现象。

### 🍎 macOS(Apple Silicon)

**下载:`MoleWorld_v0.0.2_beta_macOS_AArch64.zip`**

1. 解压 zip。
2. **右键**点击「摩尔庄园.app」→ 选择「**打开**」(请勿直接双击)。
3. 首次会弹出未签名提示,在弹窗里再点一次「**打开**」即可通过 Gatekeeper。
4. 进入游戏 🎮

> 如果右键打开仍被拦截,可在终端执行以下命令解除隔离属性后再打开:
> ```bash
> xattr -dr com.apple.quarantine 摩尔庄园.app
> ```

---

### 🪟 Windows(10 / 11 · 64 位)

**下载:`MoleWorld_v0.0.2_beta_Windows_X86_64.zip`**

1. 解压 zip。
2. 双击 **`Run-MoleWorld.bat`**。
3. 进入游戏 🎮

> 若启动后闪退:bat 脚本末尾保留了 `pause`,窗口会停住并显示报错信息,方便你截图反馈(见下方「🐞 反馈」)。

---

### 🐧 Linux(64 位桌面)

**下载:`MoleWorld_v0.0.2_beta_Linux_X86_64.tar.gz`**

1. 解压,得到**一个顶层文件夹**。
2. 进入该文件夹,双击「**启动游戏.sh**」:
   - **KDE / XFCE**:弹窗里选择「**运行**」。
   - **GNOME**:双击 `.sh` 默认只会用文本编辑器打开,请**右键 → 「以程序运行 / Run as a Program」**。
3. 或者直接在终端里运行:`./启动游戏.sh`
4. 进入游戏 🎮

> 想要桌面 / 应用菜单图标?运行同目录下的「**安装到应用菜单.sh**」即可。
> 运行环境需系统具备 **OpenGL / SDL2** 相关运行库。文件夹内还附带「**如何运行.txt**」。

### 🤖 Android(arm64)

**下载:`MoleWorld_v0.0.2_beta_Android_AArch64.apk`**

1. 把 apk 传到手机并安装(debug 签名,需要在系统设置里允许「**未知来源 / 安装未知应用**」)。
2. 打开 App,**直接进入游戏**——游戏本体已内置,无需额外下载 🎮

## ✨ 本版更新(v0.0.2 beta)

- ✨ **离线改名(含中文)**:拦截改名的服务器校验,改为**本地生效 + 落盘**(参考 2.4.3 旧版的本地改名路径)——离线也能改庄园名字、存进存档、退出重进保持。
- 🐛 **改名输入闪退修复**:输入框收到多字节字符(中文 / emoji / 粘贴)时的硬断言崩溃(`assert_eq!(len, 1)`,只允许单字符)已修,现支持任意长度输入。
- 🐛 **中文改名归档崩溃修复**:touchHLE 的 `NSString` 归档器原本写死「只支持 ASCII」(`// TODO` 占位),归档中文名直接崩;本版补上 **UTF-8 支持**(底层二进制 plist 写入器本就支持 UTF-8),与真实 iOS 行为一致。
- ⌨️ **T 键编辑冲突修复**:正在输入文字时按 `T` 不再误弹修改器菜单(改为只有非编辑状态才呼出菜单),名字里带 “t” 也能正常输入。
- 📖 **自定义「帮助 / 关于」页**:游戏内「设置 → 帮助」改为本移植项目的关于页(暖黄底,动态显示 CPU / GPU / 分辨率 / 系统时间时区、贡献者、版本与版权)。
- 🩺 **崩溃诊断增强**:日志开头新增「**系统信息块**」(移植版本 / 游戏 / 操作系统 / CPU / GPU / 时间),崩溃前留「**运行足迹**」面包屑,Windows 原生崩溃也会落盘 —— 出问题更好定位、更快修。
- 🔇 **日志降噪**:`alGetError` / `alcGetError` 只在真有错误时记录,不再刷屏 `=> 0x0`;离线分析上报也只记首条,日志更清爽。

## ⚠️ 已知限制

以下内容属于**离线复活的天然限制**(原本依赖已死的服务器),**并非 bug**:

- **17 个季节联网活动**:美术资源从服务器下载、本地不存在且需联网校验,无法进入。
- **神算子**:5.5.0 改为联网扭蛋玩法,离线不可用。
- **多人 / 好友 / 社交 / 排行榜 / 广告**:依赖服务器,离线不可用。
- **「等级不涨」**:5.5.0 新增的 `curLevel` XOR 混淆在运行时层处理仍有问题,可用**修改器(按 `T`)直接设定等级**绕过。
- **内购已停**:官方内购已停服,贝壳已改为**本地免费发放**。
