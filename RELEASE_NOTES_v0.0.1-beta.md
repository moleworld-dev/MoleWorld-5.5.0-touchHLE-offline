# 🎉 摩尔庄园HD 移动版 v0.0.1 beta · 六一儿童节快乐版本

> 还记得那只圆滚滚的小摩尔吗?那个能种地、钓鱼、装修小屋、和朋友一起守护拉姆的夏天?

2015 年,《摩尔庄园移动版 / 豪华版》(2D 平面模拟经营老游戏)随着官方服务器关闭而停运下架,从此再也登不进去。这是一份写给童年的存档:借助开源模拟器 **touchHLE**,我们让这款 32 位 ARMv7 的 cocos2d-iphone 老游戏在现代设备上**离线复活**——服务器虽然已经死了,但庄园还在。

本次首个公开 beta 覆盖 **macOS / Windows / Linux / Android 四大平台**,全部**开箱即玩**:游戏本体已内置在包里,**无需服务器、无需登录、无需任何额外下载**,解压(或安装)就能直接进庄园。

祝每一位曾经的小摩尔,六一儿童节快乐 🎈

---

## 📦 各平台下载与玩法

> 桌面三端均为**未签名**版本,首次启动时系统可能拦截,按下面步骤选择"信任 / 打开"即可,属正常现象。

### 🍎 macOS(Apple Silicon)

**下载:`MoleWorld_v0.0.1_beta_macOS_AArch64.zip`**

1. 解压 zip。
2. **右键**点击「摩尔庄园.app」→ 选择「**打开**」(请勿直接双击)。
3. 首次会弹出未签名提示,在弹窗里再点一次「**打开**」即可通过 Gatekeeper。
4. 进入游戏 🎮

> 如果右键打开仍被拦截,可在终端执行以下命令解除隔离属性后再打开:
> ```bash
> xattr -dr com.apple.quarantine 摩尔庄园.app
> ```

---

### 🪟 Windows(10 / 11 · 64 位)

**下载:`MoleWorld_v0.0.1_beta_Windows_X86_64.zip`**

1. 解压 zip。
2. 双击 **`Run-MoleWorld.bat`**。
3. 进入游戏 🎮

> 若启动后闪退:bat 脚本末尾保留了 `pause`,窗口会停住并显示报错信息,方便你截图反馈(见下方「🐞 反馈」)。

---

### 🐧 Linux(64 位桌面)

**下载:`MoleWorld_v0.0.1_beta_Linux_X86_64.tar.gz`**

1. 解压,得到**一个顶层文件夹**(不会再散落一地文件)。
2. 进入该文件夹,双击「**启动游戏.sh**」:
   - **KDE / XFCE**:弹窗里选择「**运行**」。
   - **GNOME**:双击 `.sh` 默认只会用文本编辑器打开,请**右键 → 「以程序运行 / Run as a Program」**。
3. 或者直接在终端里运行:
   ```bash
   ./启动游戏.sh
   ```
4. 进入游戏 🎮

> 想要桌面 / 应用菜单图标?运行同目录下的「**安装到应用菜单.sh**」即可。
> 运行环境需系统具备 **OpenGL / SDL2** 相关运行库。
> 文件夹内还附带「**如何运行.txt**」,看一眼就懂。

---

### 🤖 Android(arm64)

**下载:`MoleWorld_v0.0.1_beta_Android_AArch64.apk`**

1. 把 apk 传到手机并安装(debug 签名,需要在系统设置里允许「**未知来源 / 安装未知应用**」)。
2. 打开 App,**直接进入游戏**——游戏本体已内置,无需额外下载 🎮

## 📝 通用说明

- **修改器菜单**:游戏内按下 **`T`** 键即可召出修改器菜单(贝壳、经验、解锁、一键收获等)。
- **存档位置**:存档保存在各平台对应的**用户数据目录**中,卸载或清理前请注意备份。
- **未签名提示**:桌面三平台(macOS / Windows / Linux)均**未签名**,系统弹出拦截时,选择「信任 / 仍要运行 / 打开」即可。

## ✨ 本版更新(v0.0.1 beta)

作为**首个公开 beta**,本版打包并修复了以下能力:

- 🎁 **四平台开箱即玩打包**:macOS / Windows / Linux / Android 全部内置游戏本体,无需服务器 / 登录 / 额外下载。
- 🐛 **Windows NVIDIA 闪退修复**:移除了错误的非默认 VAO 绑定,解决 N 卡机器启动即崩溃。
- 🐛 **Windows 第二处闪退修复**:`/tmp` 诊断路径硬编码导致的崩溃,改为**环境变量门控、默认关闭**。
- 🐧 **Linux 一键启动器修复**:打包为**顶层文件夹** + 「启动游戏.sh」+ 「安装到应用菜单.sh」+ 「如何运行.txt」,彻底解决"双击不启动 / 解压一地散文件 / 不知道怎么启动"的问题。
- 🤖 **Android 完整内置游戏点击即玩** + 高清应用图标。
- 🎨 **离线美术修补**:从 2.4.3 旧版本反向移植了 **11 个缺失的 BGM / 音效** + **成就奖章图集**,补齐离线包缺失资源。
- 🛠️ **内置修改器菜单**:按 `T` 呼出,支持贝壳 / 经验 / 解锁 / 一键收获等。
- 🐛 **进游戏存档解档崩溃修复**:修复了读档阶段反序列化导致的崩溃。
- 🐚 **商店改为免费贝壳**:已停服的内购改为**本地免费发放**,贝壳自由。
- 💬 **旧式 UIView 弹窗 / 转场动画**:补齐老游戏的弹窗与场景切换动画表现。

## ⚠️ 已知限制

以下内容属于**离线复活的天然限制**(原本依赖已死的服务器),**并非 bug**:

- **17 个季节联网活动**:这些活动的美术资源在运行时从服务器下载、本地并不存在,且需要联网校验,因此无法进入。
- **神算子**:5.5.0 版本已改为联网扭蛋玩法,离线状态下不可用。
- **多人 / 好友 / 社交 / 排行榜 / 广告**:均依赖服务器,离线不可用。
- **「等级不涨」**:5.5.0 新增的 `curLevel` XOR 混淆在运行时层的处理存在问题。可用**修改器(按 `T`)直接设定等级**绕过。
- **内购已停**:官方内购已停服,贝壳已改为**本地免费发放**。
