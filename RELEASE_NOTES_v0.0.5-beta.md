> 这一版以**稳定性**为主线:**困扰大量摩友的「种地 / 收菜必闪退」彻底修好了**(实测复现已消除);同时把游戏本体**换回未魔改的原版(香草)基底**,所有破解 / 作弊功能改成**修改器里可逐项开关**;修改器菜单也做了**大改版**——数值实时读取、加减按钮、底部提示、危险操作二次确认,并补进一大批新挖到的作弊 / 调试选项。

## 🚨 重磅修复:种地 / 收菜闪退(很多人遇到的那个)

不少摩友反馈:**种好地、去收菜的时候必定闪退**,有时再次进游戏还会卡死进不去。这一版**从根上修好了**。

- **根因**(技术向,可跳过):存档归档时 `NSKeyedArchiver` 按对象指针去重的表没被及时清掉,导致 `npcs / 成就解锁列表 / 属性值`**三个本不相干的字段塌缩成了同一个数组** —— 于是成就永远「未解锁」,每次收菜重复发奖、内存爆掉闪退;损坏的存档再进游戏还会触发死循环卡住。
- **修法**:归档期对参与去重的对象**临时持有(retain)、归档结束再释放**(治本);叠加两层兜底(空操作防御 + 坏档时抑制成就重复触发)。
- **结果**:**反复种地→收菜不再闪退**,已有的坏档也能正常进入。

> 如果你之前的存档已经被旧版本写坏了,本版能正常读取;若仍异常,可用修改器(按 `T`)→「开发者 / 调试」页 →「⚠️ 重置本地存档」一键清档重来。

## ✨ 本版更新(v0.0.5 beta)

### 🛡️ 换回原版(香草)基底,破解功能全部「可开关」

- 以前的包是基于**第三方魔改版(无限贝壳)**,容易数据溢出、出奇怪问题。本版**换回了官方原版二进制脱壳后的「香草」基底**,更贴近原作、更稳。
- 原本魔改版里写死的那些破解(无限贝壳树 / 免费贝壳墙 / 免费占卜 / 节日村进入 / 商城免 VIP / 跳对象校验 / 去越狱检测……)**全部改成了修改器里可逐项开关的功能**,运行时按需打补丁。**默认只开必要的几项**(如「修复占卜」「进新岛门护黄金岛」),其余想用再自己开,干净可控。

### 🎛️ 修改器(按 `T`)大改版

- **菜单重构为 5 个分页**:`数值` / `召唤` / `小游戏·任务·重置` / `开关·解锁·成就` / `开发者·调试`。
- **数值实时读取 + 加减按钮**:经验 / 摩尔豆 / 贝壳等**直接读取你存档里的当前值**显示,配 `+1 / +10 / +100 / -1 / -10` 等快速按钮稳定增减(等级上限 52),不再是以前那个点不准、改不动的滑块。
- **底部状态提示(toast)**:每次开关功能,菜单底部会提示「XX 已开启 / 已关闭」,执行结果一眼可见。
- **危险操作二次确认**:重置存档、覆盖家园等不可逆操作,**需要再点一次确认**才执行,防误触。

### 🆕 新增 / 补全的作弊 · 解锁 · 调试

- 🔓 **全解锁补全**:作物 / 物品 / 装扮 / 音乐 / 头像等锁判定补齐,VIP 专属头像也免 VIP。

## 📦 各平台下载与玩法

<details>
<summary>👉 <b>点击展开</b> —— macOS / Windows / Linux / Android 各平台下载与一键玩法</summary>

> 桌面三端均为**未签名**版本,首次启动时系统可能拦截,按下面步骤选择「信任 / 打开」即可,属正常现象。

### 🍎 macOS(Apple Silicon)

**下载:`MoleWorld_v0.0.5_beta_macOS_AArch64.zip`**

1. 解压 zip。
2. **右键**点击「摩尔庄园.app」→ 选择「**打开**」(请勿直接双击)。
3. 首次会弹出未签名提示,在弹窗里再点一次「**打开**」即可通过 Gatekeeper。
4. 进入游戏 🎮(想锁比例 / iPhone 版,改用解压目录里对应的 `.command` 启动脚本)

> 如果右键打开仍被拦截,可在终端执行:`xattr -dr com.apple.quarantine 摩尔庄园.app`

---

### 🪟 Windows(10 / 11 · 64 位)

**下载:`MoleWorld_v0.0.5_beta_Windows_X86_64.zip`** → 解压 → 双击 **`Run-MoleWorld.bat`** → 进游戏 🎮

### 🐧 Linux(64 位桌面)

**下载:`MoleWorld_v0.0.5_beta_Linux_X86_64.tar.gz`** → 解压进文件夹 → 双击 **`启动游戏.sh`**(GNOME 需右键「以程序运行」)→ 进游戏 🎮。需 OpenGL / SDL2 运行库。

## ⚠️ 已知限制

以下内容属于**离线复活的天然限制**(原本依赖已死的服务器),**并非 bug**:

- **17 个季节联网活动**:美术资源从服务器下载、本地不存在且需联网校验,无法进入。
- **黄金岛(加勒比寻宝)**:核心美术绝大部分是服务器下载的、本地及旧版全无,离线仍进不去(点了干净退回村庄);可建筑新岛的离线复活方案仍在攻坚,相关实验性开关暂只放在「开发者 / 调试」页,供进阶玩家折腾。
- **多人 / 好友 / 社交 / 排行榜 / 广告 / VIP 在线奖励 / 充值**:依赖服务器,离线不可用。
- **「等级不涨」**:5.5.0 新增的 `curLevel` 混淆在运行时层处理仍有问题,可用**修改器(按 `T`)直接设定等级**绕过。
- **retina 高清美术**:模拟器 `UIScreen.scale` 暂写死 1.0,游戏只加载非 retina 美术(iPad 1024×768 是当前最清晰路径);retina 支持是后续独立课题。
