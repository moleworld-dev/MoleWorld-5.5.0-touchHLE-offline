摩尔庄园HD 移动版 · 现代化离线移植(基于 touchHLE)
Linux x86_64 · v0.0.1 beta
================================================================

游戏已内置,无需联网、无需登录、无需额外下载。
三种启动方式,任选其一:

【方式 1 · 终端(最稳)】
  打开终端,cd 进入本文件夹,执行:
      ./启动游戏.sh
  若提示没有权限:先执行
      chmod +x 启动游戏.sh touchHLE
  再重新运行。

【方式 2 · 双击「启动游戏.sh」】
  · KDE(Dolphin)/ XFCE(Thunar):双击 → 选「运行 / Run」。
  · GNOME(文件 / Nautilus):双击 .sh 默认只会用文本编辑器打开它,
    请改为【右键 →「以程序运行 / Run as a Program」】。

【方式 3 · 生成桌面/菜单图标(GNOME 推荐,最省心)】
  双击或在终端运行同目录的:
      安装到应用菜单.sh
  之后在应用程序列表里搜「摩尔」,即可像普通软件一样点击启动。

----------------------------------------------------------------
游戏内操作:
  · 鼠标左键 = 触摸(点击 / 拖动)
  · 按 T 键   = 召出修改器菜单(贝壳 / 经验 / 解锁 / 一键收获…)
  · 退出      = 关闭游戏窗口,或在终端按 Ctrl+C

系统要求:
  · 64 位桌面 Linux,需有 OpenGL / SDL2 运行库,以及 X11 或 Wayland 桌面。
  · 若启动时报缺少 libGL / libasound 等,请用发行版包管理器安装对应运行库
    (例如 Debian/Ubuntu: sudo apt install libgl1 libasound2 libpulse0)。

遇到问题:
  请把终端里的报错信息截图,提交到 GitHub Issues:
  https://github.com/Shad0w23333/MoleWorld-5.5.0-touchHLE-offline/issues
